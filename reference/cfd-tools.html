Ruled out (and why)

  • Fusion 360 Sim Extension — structural/thermal/molding only. No external-flow CFD at all. Use Fusion to model, not simulate flow.
  • Autodesk CFD — still exists, but ~$7,295/yr, no hobbyist tier, separate app from Fusion.
  • SolidWorks Flow Simulation — real CFD but only inside SolidWorks; no Fusion fit; quote-only.
  • Flow Illustrator / browser toys — qualitative “artist impression” only; no quantitative lift/drag.
  • Possible 4th if on Windows: Ansys Student — free, 512k-cell cap; Discovery Live great for instant qualitative ranking. Heavier than the three above to start.
Before you trust any CFD number
CFD produces confident-looking results that can be badly wrong. Learn y+ / near-wall meshing, mesh-independence, and convergence first: Fluid Mechanics 101 (Wimshurst) — Getting Started, y+. Validate your runs against the NASA NACA 0012 case; formalise mesh studies with the Grid Convergence Index (GCI / ASME V&V 20).
