# Hydrodynamic Foils

A learn-by-building course on the hydrodynamics of underwater foils — aimed at designing, computing, 3D-printing, and water-testing fishing **depressors** (towed downforce wings / diving planers) for trolling at 3–10 kn.

**▶ Live site:** https://chrisfraser.github.io/foil-design/

## Contents
- `index.html` — course landing page
- `lessons/` — self-contained HTML lessons (start at `0001-foils-101.html`)
- `reference/` — quick-reference docs (glossary)
- `assets/` — shared stylesheet + quiz widget
- `MISSION.md` — why this course exists / success criteria
- `RESOURCES.md` — vetted knowledge sources and communities
- `learning-records/` — what's been learned (drives what's taught next)
- `NOTES.md` — teaching notes & learner preferences

Built as a [Claude Code `/teach`](https://claude.com/claude-code) workspace.
